## Installation

BlockIDSDK is available through CocoaPods. To install it, simply add the following line to your Podfile:

```ruby
  pod 'BlockIDSDK', :git => 'https://<user>:<token>@github.com/1KBlockID/ios-blockidsdk.git', :tag => '1.8.02'
```
